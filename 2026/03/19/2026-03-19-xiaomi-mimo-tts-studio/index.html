Xiaomi MiMo V2.5 语音合成实验室：多模型对比试听

发表于2026-03-19|更新于2026-04-30|技术分享
|浏览量:

小米近期升级了 MiMo-V2.5-TTS 系列语音合成模型。新版支持预置精品音色、文本设计音色、音色复刻三种路径。为了方便大家体验，我把在线实验室升级为多模型对比模式，可以一次生成两路语音并排试听。

🚀 立即体验 MiMo TTS

你可以在下方文本框中输入任何想要合成的文字，选择两个模型进行对比，并尝试使用“快捷风格调节”来赋予声音不同的个性（如：开心、粤语、唱歌模式）。

MiMo TTS 2.5 对比控制台

一次选择两个模型，并排生成语音，快速听出差异。

V2.5

为控制等待时间和费用，页面限制同时生成两个结果。

仅在选择音色复刻时需要，文件 Base64 后不超过 10 MB。

快捷风格调节
合成失败，请检查设置或 API 状态。

技术细节

  • 模型：mimo-v2.5-tts、mimo-v2.5-tts-voicedesign、mimo-v2.5-tts-voiceclone
  • 能力：支持预置音色、文本设计音色、音色复刻，以及细粒度风格标签。
  • 实现：采用 Vanilla JS 与 Fetch API，并排输出两个播放器，方便快速 AB 对比。

大家如果在使用中遇到任何问题，欢迎在下方留言交流！
