Xiaomi MiMo V2 语音合成实验室：让你的文字动起来

发表于2026-03-19|更新于2026-03-19|技术分享
|浏览量:

小米近期正式上线了 MiMo-V2-TTS 全新语音合成模型。相比前代，V2 版本在自然度、情感表达以及风格控制上有了质的飞跃。为了方便大家体验，我基于其开放 API 开发了这个简单直观的在线实验室。

🚀 立即体验 MiMo TTS

你可以在下方文本框中输入任何想要合成的文字，并尝试使用“快捷风格调节”来赋予声音不同的个性（如：开心、粤语、甚至唱歌模式）。

技术细节

  • 模型：mimo-v2-tts
  • 能力：支持细粒度的控制标签（如 [沉默片刻]、呼—— 等呼吸音）。
  • 实现：采用 Vanilla JS 与 Fetch API，确保在各种博客主题下均能流畅运行。
